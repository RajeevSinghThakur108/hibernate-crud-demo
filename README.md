Hibernate CRUD Demo Project Documentation
Project Overview
This project demonstrates basic CRUD (Create, Read, Update, Delete) operations using Hibernate with StandardServiceRegistry and a MySQL database. The project is designed to be simple and straightforward, suitable for beginners looking to understand Hibernate's core functionalities.

## Requirements
Software Requirements
Java: Version 8 or higher
Maven: For building and managing the project
MySQL: Database server to store and manage data

## Dependencies
The project relies on the following Maven dependencies:
Hibernate Core: Version 6.4.8.Final
MySQL Connector: Version 8.0.21

## Database Setup
Database Name: hibernate_demo
User: root
Password: password

## Functionality
1. Create a New Student
Allows adding a new student record to the database with details such as name and city.

2. Read Students
Fetches and displays all student records from the database. Each student's name and city are printed to the console.

3. Update a Student
Updates an existing student's details, such as changing the city based on the student's ID.

4. Delete a Student
Deletes a student record from the database based on the student's ID. Ensures that the student is removed from the database.

## Conclusion
This project provides a foundational understanding of how to use Hibernate for basic CRUD operations. It includes the necessary setup, configurations, and implementation details to get started with Hibernate and MySQL. By following the usage instructions, you can set up the project, run it, and observe how Hibernate interacts with the database to perform CRUD operations.
